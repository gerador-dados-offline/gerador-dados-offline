# Gerador de Dados Offline (CPF, CNPJ e E-mail)

Ferramenta 100% offline para gerar CPFs, CNPJs e e-mails válidos para testes, com exportação CSV, tema escuro/claro e opções por região (CPF).

- Geração em lote (até 1000 itens)
- Pontuação opcional (CPF/CNPJ)
- E-mail com domínio e usuário personalizados
- Exportação CSV com fallback (nova aba)
- 100% no navegador (sem servidores)

## Deploy
- GitHub → Vercel (Static Site)
- Domínio customizado opcional

## Privacidade
Nenhum dado é enviado a servidores. Tudo roda localmente no navegador.

# Offline Data Generator (CPF, CNPJ, and Email)

100% offline tool to generate valid CPFs, CNPJs, and test emails, with CSV export, dark/light theme, and regional options (CPF).

- Batch generation (up to 1000 items)
- Optional punctuation (CPF/CNPJ)
- Custom email user and domain
- CSV export with fallback (new tab)
- Runs entirely in the browser

## Deployment
- GitHub → Vercel (Static Site)
- Optional custom domain

## Privacy
No data is sent to servers. Everything runs locally in the browser.
